# Unitools starter

Unitools is a group of tooling that aims to simplify the process of code sharing between Next.js, Expo, and React Native projects. Itâ€™s a required and indispensable tool in the realm of cross-platform development today with more and more apps moving towards universal app development. Unitools provides a unified API for handling images, links, and route navigation hooks across all these platforms.

## Installation

```
yarn
```

### Run Expo app

```
yarn run:expo
```

### Run Next.js app

```
yarn run:next
```
